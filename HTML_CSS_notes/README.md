Notes from "Front-end foundations" course, including:

* Basic structure of a page.

* Ordered and unordered lists: `<ol>` and `<ul>` (containing `<li></li>` child tags).

*CSS selectors.

* The box model: `content`, `padding`, `margin`, `border`
